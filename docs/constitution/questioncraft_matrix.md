# Questioncraft Matrix — Constitutional Reference

The Questioncraft Matrix is Owlume’s core clarity framework.

Its purpose is to structure inquiry, not to produce answers.
It exists to help users see blind spots in their thinking by asking better questions at the right moment.

The Matrix is composed of Modes and Principles.
Each Mode represents a distinct way of examining a situation.
Each Principle represents a specific angle of scrutiny within that Mode.

The Matrix does not encode conclusions, advice, or correct outcomes.
It encodes perspectives that are commonly missed when decisions are made under pressure, uncertainty, or cognitive load.

The Matrix is used to:
- reveal hidden assumptions
- surface unexamined constraints
- test confidence against uncertainty
- distinguish framing errors from information gaps
- slow premature closure

The Matrix is not:
- a doctrine
- a checklist to be completed
- a scoring system
- a decision rule
- a substitute for judgment

The Matrix does not tell the user what to think or what to do.
It helps the user see where their thinking may be incomplete.

The value of the Matrix lies in its structure.
It ensures that important dimensions of a decision are not skipped simply because an answer is easy to generate.

The Matrix serves judgment.
It never replaces it.
