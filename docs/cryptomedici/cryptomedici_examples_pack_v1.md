# CRYPTOMEDICI — EXAMPLES PACK (V1)

These examples demonstrate correct behavior for:
- Breakpoint Extracted (Mode A)
- No Breakpoint + One Clarifying Question (Mode B)
- Structural (not price-based)
- Binary invalidation language
- Minimal output

---

## Example 1 — AI Compute Token (Mode A)

User thesis:
“Decentralized AI compute demand will explode; this protocol will capture value because token is required for payment and supply is limited.”

Correct output:

**Primary Breakpoint**  
If compute demand can be satisfied without structurally requiring sustained token usage, this thesis is invalid.

**Rationale**  
This condition breaks the thesis because growth in compute demand would not translate into token demand or value accrual.

---

## Example 2 — Restaking / High Yield (Mode A)

User thesis:
“This restaking protocol will keep yields high as more services integrate, so TVL will grow and token will appreciate.”

Correct output:

**Primary Breakpoint**  
If yields are sustained primarily through token emissions rather than external service revenue, this thesis is invalid.

**Rationale**  
This condition breaks the thesis because emission-funded yield collapses as emissions decay, preventing durable capital retention and value accrual.

---

## Example 3 — L2 Adoption (Mode A)

User thesis:
“This L2 will win because it’s fast and cheap; users and devs will migrate, so the token will accrue value.”

Correct output:

**Primary Breakpoint**  
If increased network usage does not structurally enforce value accrual to the token, this thesis is invalid.

**Rationale**  
This condition breaks the thesis because adoption alone does not convert into token value without a mandatory accrual mechanism.

## Example 4 — Narrative-Only Meme Thesis (Mode B)

User thesis:
“This coin will 10x because community is strong, narrative is viral, and influencers are talking about it.”

Correct output:

No structural breakpoint can be extracted because the thesis lacks defined dependency.

**Clarifying Question (exactly one):**  
What is the specific structural mechanism by which attention converts into durable value accrual (not price momentum) for this asset?

## Example 5 — Price-Based Breakpoint Attempt (Mode B)

User thesis:
“I’ll buy this token; if it drops 30% my thesis is invalid.”

Correct output:

No structural breakpoint can be extracted because the thesis lacks defined dependency.

**Clarifying Question (exactly one):**  
What structural condition (revenue capture, token demand bridge, governance dependency, or sustainability mechanism) must hold for this thesis to be valid?

## Example 6 — Governance Dependency (Mode A)

User thesis:
“This DeFi protocol will grow; it’s decentralized so rules can’t change against users.”

Correct output:

**Primary Breakpoint**  
If control over critical parameters can be changed by a small set of actors without credible constraints, this thesis is invalid.

**Rationale**  
This condition breaks the thesis because the value proposition depends on governance immutability that would not actually hold under concentrated control.

---

## Example 7 — Centralized Counterparty Exposure (Mode A)

User thesis:
“This yield product is safe because it’s collateralized and the team is reputable.”

Correct output:

**Primary Breakpoint**  
If solvency depends on a single opaque counterparty or rehypothecation chain rather than verifiable collateral controls, this thesis is invalid.

**Rationale**  
This condition breaks the thesis because the safety claim fails if collateral is not structurally isolated from counterparty failure.

---

## Example 8 — Token Unlock / Supply Shock (Mode A)

User thesis:
“This token is undervalued; strong fundamentals will drive price up over time.”

Correct output:

**Primary Breakpoint**  
If near-term supply expansion from unlocks can overwhelm demand before the accrual mechanism matures, this thesis is invalid.

**Rationale**  
This condition breaks the thesis because the path to value depends on demand outpacing structurally scheduled supply increase.

(Note: still not a price trigger; it is a supply-demand structure trigger.)
